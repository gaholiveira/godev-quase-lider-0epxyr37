# High-Ticket Launch

Landing page for high-ticket launches with sections for Hero, Social Proof Bar, Manifesto, Product Showcase, and Testimonials.

## Quick Start

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Features

- ✅ Hero section with CTA
- ✅ Social proof metrics bar
- ✅ Manifesto/mission statement
- ✅ Product showcase with pricing
- ✅ Customer testimonials
- ✅ Responsive design (mobile-first)
- ✅ Zinc palette (high-end aesthetic)
- ✅ Luxury Motion (hover effects, transitions)
- ✅ TypeScript strict mode
- ✅ Vercel-ready

## Tech Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS
- TypeScript
- Lucide React (icons)

## Deploy

Push to GitHub and import in [Vercel](https://vercel.com). Zero config needed.
